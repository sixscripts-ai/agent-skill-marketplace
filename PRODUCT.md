# Product

## Register

product

## Users
AI Engineers and developers building, managing, and deploying autonomous coding skills. They operate in a highly technical context where precision, control, and efficiency are paramount.

## Product Purpose
To serve as a functional tool, dashboard, and workspace for creating and managing agentic skills. Success looks like an interface that instills expert confidence, precision, and state-of-the-art power without getting in the way.

## Brand Personality
Precise, Technical, Uncompromising.

## Anti-references
- "Ghost-card" SaaS templates, over-rounded elements (32px+ radius), decorative gradients/stripes, and "fluffy" marketing designs.
- Dense data-tables, brutalist plain HTML, zero spacing.

## Design Principles
1. **Expert Confidence**: The UI should feel like a professional IDE or pro-level tool, not a consumer app.
2. **Precision over Decoration**: Every visual element must serve a functional purpose.
3. **Show, Don't Tell**: Information density should be high but structured cleanly. 
4. **Practice what you Preach**: As an AI coding tool, the interface itself should be a state-of-the-art example of modern web development.

## Accessibility & Inclusion
Standard WCAG AA compliance with robust keyboard navigation and reduced-motion support.
